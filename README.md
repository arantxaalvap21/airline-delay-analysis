# Credit Risk Analytics

## Descripción

Proyecto de Business Intelligence enfocado en el análisis de riesgo crediticio utilizando AWS Aurora PostgreSQL, Python y Looker Studio.

## Problema de negocio

Las instituciones financieras necesitan identificar factores asociados al incumplimiento de créditos para mejorar la evaluación de riesgo y la toma de decisiones.

## Objetivo

Desarrollar una solución de análisis de riesgo crediticio que incluya:

- Proceso ETL en Python
- Base de datos Aurora PostgreSQL
- Modelo dimensional tipo estrella
- Dashboard interactivo en Looker Studio

## Tecnologías

- AWS Aurora PostgreSQL
- Python
- SQL
- Looker Studio
- GitHub

## Arquitectura propuesta

Dataset → ETL Python → Aurora PostgreSQL → Modelo Estrella → Looker Studio
